The product

Make 3D creation immediate, competitive and shareable.

Sculpt Showdown is a multiplayer browser experience built around voxel sculpting. The project combines a real-time interaction model with custom tooling for creating and manipulating block-based forms.

Engineering focus

Build the tooling beneath the experience.

  • Use a browser-native 3D stack. Three.js keeps the experience accessible without a desktop install.
  • Create custom voxel tooling. The interaction model is built around the product rather than a generic editor.
  • Ship continuously. Vite and Cloudflare Pages support a tight build-and-deploy loop.
  • Polish before scale. Current work is focused on visual quality ahead of public playtesting.
Current status

The project is deployed and undergoing visual polish. Player-count, session and playtest results will be added when public testing begins.
