Project 02 / Autonomous systems

ADA²M Drone

A two-person autonomous drone project for collecting temperature, humidity and air-quality data over farmland.

RoleMechanical + autonomy
TeamTwo people
Timeline2025 - present
StatusFlight testing
The brief

Collect useful environmental data without requiring manual field coverage.

ADA²M is designed to sweep predefined flight paths over farmland and log environmental measurements for small-scale farmers. The engineering problem combines airframe packaging, flight control, long-range communication, onboard compute and sensor integration.

Architecture

A compact autonomous stack.

Flight pathPredefined agricultural coverage route.
Flight controlSpeedyBee F405 V4 stabilises and commands the aircraft.
Onboard intelligenceJetson Nano runs inference for flight decisions.
Environmental dataTemperature, humidity and air-quality measurements.
AirframeCustom carbon-fibre frame
Long-range linkTBS Crossfire
Design toolSolidWorks
SoftwareC++ / Python
Engineering work

Build the physical platform and make it behave.

  • Package the flight stack. Design the custom carbon-fibre frame around propulsion, control and compute requirements.
  • Connect autonomy to flight. Integrate Jetson Nano inference with onboard decision-making.
  • Design for range. Use a TBS Crossfire link for long-range control and testing.
  • Test progressively. Move from partial flight to stability tuning before expanding sensor and autonomy capability.
Current status

The aircraft is airborne and in flight-stability tuning, with sensor integration continuing. Quantified endurance, payload and coverage results will be added after the test program produces reliable data.
