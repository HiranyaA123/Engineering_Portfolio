# Hiranya Agarwal - Engineering Portfolio

Static portfolio website for Hiranya Agarwal, a Mechanical Engineering (Honours) student at UNSW working across robotics, autonomous systems, space hardware and practical software.

## Local preview

Open `index.html` in a browser. No build step is required.

## Deployment

The workflow in `.github/workflows/deploy-pages.yml` deploys the repository root to GitHub Pages whenever `main` is updated. In the repository settings, set **Pages > Build and deployment > Source** to **GitHub Actions**.

## Structure

- `index.html` - portfolio landing page
- `projects/` - six detailed project case studies
- `styles.css` and `enhancements.css` - responsive visual system
- `Hiranya_Agarwal_Resume.pdf` - linked one-page resume
